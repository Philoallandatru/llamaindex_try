LlamaIndex 配置管理系统测试文档

这是一个用于测试新实现的配置管理系统的文档。

主要功能：
1. 数据源管理 - 支持 File、Jira、Confluence 等多种数据源类型
2. 知识库管理 - 可以关联多个数据源，构建统一的知识索引
3. 模型配置 - 支持配置多个 OpenAI-compatible 的 LLM 模型
4. Chat 增强 - 支持在对话时动态选择知识库和模型

技术栈：
- FastAPI: Web 框架
- LlamaIndex: RAG 引擎
- ChromaDB: 向量数据库
- MinerU: 文档解析

测试场景：
本文档将被上传到系统，创建为 File 类型的数据源，然后关联到知识库中。
通过同步操作，文档内容将被索引到向量数据库。
最后通过 Chat API 验证能否正确检索到文档内容。
