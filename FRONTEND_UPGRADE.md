# Frontend UI/UX Upgrade - GPT Style

## 概述

将前端界面升级为现代化的 ChatGPT 风格设计，提供专业的用户体验。

## 完成的改进

### 1. 视觉设计升级

#### 配色方案
- **浅色模式**：清新的白色和浅灰色调
- **深色模式**：优雅的深色背景，自动适配系统主题
- **强调色**：使用 `#19c37d` 绿色作为主色调（类似 ChatGPT）
- **渐变效果**：用户头像使用渐变背景

#### 组件样式
- **圆角设计**：统一使用 8-12px 圆角
- **阴影效果**：添加微妙的阴影增强层次感
- **边框**：使用 1.5px 边框提升精致度
- **过渡动画**：所有交互元素添加平滑过渡

### 2. 聊天界面改进

#### 空状态设计
```tsx
- 欢迎标题："How can I help you today?"
- 副标题说明系统功能
- 4 个建议卡片，可点击快速开始对话
- 大图标增强视觉吸引力
```

#### 消息显示
- **用户消息**：浅灰色背景，渐变头像
- **AI 消息**：白色背景，带边框头像
- **最大宽度**：800px 居中显示，提升可读性
- **间距优化**：24px 消息间距，20px 内边距

#### 源引用卡片
- 悬停效果：边框变色 + 阴影
- 文本截断：最多显示 3 行
- 标题加粗，摘要使用次要文本颜色
- 圆角卡片设计

### 3. 输入区域重构

#### 配置栏
```tsx
- 知识库选择器
- 模型选择器
- 统一的下拉框样式
- 悬停和聚焦状态
```

#### 输入框
- **圆角设计**：12px 圆角
- **浮动按钮**：右下角发送按钮（↑ 图标）
- **自动调整高度**：根据内容自动扩展
- **聚焦效果**：绿色边框 + 阴影
- **占位符**："Message Knowledge Portal..."

#### 发送按钮
- 36x36px 圆形按钮
- 绿色背景，白色箭头图标
- 悬停时放大效果
- 禁用状态透明度降低

### 4. 侧边栏优化

#### 布局
- 宽度：280px
- 浅色背景（浅色模式）/ 深色背景（深色模式）
- 清晰的分隔线

#### 新建聊天按钮
- 居中对齐，带图标
- 边框样式，悬停时变色
- 字体加粗

#### 会话列表
- 文本溢出省略
- 活动状态高亮
- 平滑悬停效果

### 5. 交互体验

#### 打字指示器
```css
- 3 个跳动的圆点
- 错开的动画延迟
- 上下跳动效果
```

#### 滚动条样式
- 8px 宽度
- 圆角设计
- 悬停时颜色变化

#### 建议卡片
- 网格布局，自适应列数
- 悬停时上移 + 阴影
- 边框高亮效果

### 6. 响应式设计

- 最大内容宽度：800px
- 自动居中对齐
- 适配不同屏幕尺寸
- 深色模式自动切换

## 技术实现

### CSS 变量系统
```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f9f9f9;
  --text-primary: #2d2d2d;
  --accent: #19c37d;
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.07);
  ...
}
```

### React 组件优化
- 使用 `useRef` 管理 textarea 自动调整
- 建议卡片点击填充输入框
- 空状态条件渲染
- 平滑滚动到底部

### API 集成
- 支持知识库选择
- 支持模型选择
- 参数正确传递到后端

## 使用方法

### 启动前端
```bash
cd frontend
npm install
npm run dev
```

访问：http://localhost:5173

### 启动后端
```bash
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8002
```
