Evidence: The NVM Express Base Specification (Rev 2.1) documents standard error codes for firmware issues (e.g., "Invalid Firmware Image," "Firmware Slot Invalid"), but it does not define specific performance degradation thresholds. Conclusion: The root cause is a firmware behavior change in version 3.3 that negatively impacts I/O throughput and latency on the Samsung 980 PRO under Gen4 x4 conditions.
